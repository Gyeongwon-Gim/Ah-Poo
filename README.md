## 배포 링크

### [ah-poo.kr](https://ah-poo.kr)


## 서비스 개요

"어푸!(ah-poo!)"는 전국의 일일 입장이 가능한 수영장 정보를 쉽고 빠르게 찾아주는 **위치 기반 수영장 탐색 서비스**입니다.<br/>


## 주요 기능
* 전국의 일일 입장 가능 수영장을 탐색할 수 있어요.
* 수영장 위치를 지도 위 마커로 한눈에 확인하고, 편리하게 길을 찾을 수 있어요.
* 일일 입장 요금을 확인할 수 있어요.
* 50m 레인 유무 등 수영장 시설 규모를 미리 확인할 수 있어요.
* 자주 방문하는 수영장을 즐겨찾기에 저장할 수 있어요.
